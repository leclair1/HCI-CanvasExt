# Canvas LMS Enhancement Extension - Group Project

## What It Does

This is a **group project** hosted on **GitHub** that creates a browser extension to supercharge Canvas LMS for college students. When complete, students will be able to install the extension and instantly access AI-powered study tools, intelligent course organization, and personalized learning analytics—all without leaving Canvas. The extension adds a sleek navigation bar to Canvas with four main tabs (Home, Flashcards, Planner, Insights) plus dedicated workspaces for each course, featuring automatic flashcard generation from course materials, AI-created study plans and summaries, unified deadline tracking across all courses, performance analytics with visual charts, and full customization options including dark mode, themes, and layout preferences.

## Technologies Used

**Frontend:**
- React.js with TypeScript
- Tailwind CSS v4.0 for styling
- Shadcn/ui component library
- Recharts for data visualization
- Lucide React for icons

**Backend:**
- Node.js
- PostgreSQL database
- Canvas API integration for real-time course data synchronization

**Version Control & Collaboration:**
- GitHub for team collaboration, version control, and project management

## Core Features

### 1. **AI-Powered Study Tools**
- **Flashcard Generation**: Automatically generate flashcards from course materials using AI
- **Summary Generation**: Create intelligent summaries of lecture notes and readings
- **Quiz Creation**: AI-generated practice quizzes based on course content

### 2. **Intelligent Organization**
- **Unified Dashboard**: Centralized view of all courses, deadlines, and assignments
- **Automatic Syllabus Extraction**: Parse and extract important dates and requirements from syllabi
- **Smart Planner**: AI-powered personalized study plans based on assignments, grades, and course syllabi
- **To-Do List**: Comprehensive task management with priority levels and completion tracking

### 3. **Study Insights & Analytics**
- **Performance Tracking**: Monitor grades, quiz scores, and flashcard accuracy across courses
- **Study Pattern Analysis**: Track study time, activity distribution, and learning streaks
- **Progress Visualization**: Charts and graphs showing academic progress over time
- **Achievement System**: Gamification to encourage consistent study habits

### 4. **Quick Access Features**
- **Course Search**: Fast search functionality for finding course files and materials
- **Study Timer**: Built-in timer for tracking study sessions
- **Quick Navigation**: Fixed global navigation bar for instant access to all features

### 5. **Personalization**
- **Dark Mode**: Eye-friendly dark theme option
- **Custom Themes**: Multiple color schemes (Blue, Purple, Green)
- **Typography Controls**: Adjustable font size and font family
- **Spacing Options**: Compact, normal, or comfortable layout spacing
- **Card Styles**: Different visual styles for content cards

## Navigation Structure

### Global Navigation (Fixed Top Bar)
- **Home**: Dashboard with course overview and quick stats
- **Flashcards**: Study interface with all flashcard decks
- **Planner**: AI-generated study plans and to-do lists
- **Insights**: Analytics, performance metrics, and achievements

### Course-Specific Tabs
- **Overview**: Course details, upcoming assignments, and quick actions
- **Learn**: Access to course materials and resources
- **Plan**: Course-specific study schedule
- **Flashcards**: Course-specific flashcard decks
- **Quizzes**: Practice quizzes and assessments

## Design Principles

- **Clean & Minimal**: Uncluttered interface that integrates seamlessly with Canvas
- **HCI-Focused**: Follows human-computer interaction best practices for usability
- **Fully Responsive**: Optimized for both desktop and mobile access
- **Accessible**: Designed with accessibility in mind
- **Non-Intrusive**: Enhances Canvas without disrupting the native experience
